**What & when:** Every clickable action. `primary` (blue) for the one main action per view; `secondary` for neutral actions; `ghost` for low-emphasis; `danger`/`success`/`orange` for intent.

```jsx
<Button variant="primary"><Plus size={16} /> Add Tool</Button>
<Button variant="secondary" size="sm">Reset</Button>
<Button variant="danger">Delete</Button>
```

Lead with a 14–16px lucide icon for primary actions. One primary per screen. Sizes: `sm` for toolbars, `lg` for hero/login.
